Welcome to EEMS

Employee Expense Management System is affected by the factors like authentication. If the authentication is not give we are not allowed to use any services. The employee, manager and admin have to login to the website with his account with correct username and password. If the login is successful then he is able to use the service provided by the system software.
There are users for this system. By this system the employee can check status of his voucher, view rejected vouchers, create drafts, he can also view his profile. The manager views his profile, approves vouchers, forwards vouchers, view inbox. The admin updates employees, deletes employees and registers company.
